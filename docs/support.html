How to Use HEX

Color Picker

Drag your finger across the color grid to select a color. The hex code and RGB values update in real time. Tap the copy button to save the hex code to your clipboard.

Challenge Mode

A target color is displayed below the grid. Find it on the spectrum by dragging until the match meter reaches 100%. Track your accuracy, time, and streak across rounds.

Troubleshooting

  • App not responding: Force close and reopen the app.
  • Copy not working: Ensure the app has clipboard access in your device settings.
